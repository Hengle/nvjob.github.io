Rain Voice Note

Voice notepad for typing using voice using a microphone (Speech To Text).
This is an open source WEB application made using CW Frame (nvjob.github.io/apps/cw-frame).
Voice recognition is implemented using the Google Cloud Speech API (cloud.google.com/speech-to-text).



For Windows (Desktop App)
Current version 1.0.0.3
